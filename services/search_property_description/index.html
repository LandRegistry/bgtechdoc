Search by Property Description

Overview

Use this service to find out the title number and tenure of a property based on the address supplied.

If you’re a software developer

Use this document to integrate data into your system.

Process flow

Input

Request an official search by sending the following parameters:

  • login details
  • external reference number
  • customer reference
  • address

Validation

Validation diagram (Search by property description) details the validation that the request undergoes.

Validation diagram Business Rules
Business Rule Decision point BG Code Message ID Message text
BRL-ISBG-001 Is the address valid? N bg.invalid.property.search.criteria MSG-BG-001 Insufficient address details. Please provide flat/house and postcode OR flat, street and town OR house, street and town.
BRL-ISBG-002 Is Postcode valid? N bg.postcode.invalid MSG-BG-004 Please provide valid postcode
BRL-ISBG-003 Were matching properties found? N bg.properties.nopropertyfound MSG-BG-003 No title number has been identified from the data supplied. This does not necessarily mean that a register of a title does not exist but only that insufficient data has matched.
BRL-ISBG-003 Were too many properties returned? Y bg.properties.toomanyproperties MSG-BG-002 The property address you entered has matched with a large number of properties on our database. Please request again with refined address details.

Output

A list of properties and title numbers.
